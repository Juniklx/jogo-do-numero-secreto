# 🎲 Jogo do Número Secreto

Um jogo interativo desenvolvido com **HTML**, **CSS** e **JavaScript**, onde o objetivo é descobrir o número secreto gerado aleatoriamente pelo sistema utilizando o menor número possível de tentativas.

## 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de lógica de programação utilizando JavaScript, como:

* Estruturas de decisão;
* Laços de repetição;
* Funções;
* Manipulação do DOM;
* Geração de números aleatórios;
* Validação de entradas;
* Organização e reutilização de código.

Durante o jogo, o usuário recebe dicas indicando se o número secreto é **maior** ou **menor** que o valor informado, tornando a experiência dinâmica e intuitiva. Esse tipo de projeto é comum em cursos introdutórios de desenvolvimento web e ajuda a consolidar a lógica de programação.

## 🎮 Como jogar

1. O sistema gera um número secreto.
2. Digite um palpite.
3. O jogo informará se o número secreto é maior ou menor.
4. Continue tentando até acertar.
5. Ao final, será exibida a quantidade de tentativas realizadas.
